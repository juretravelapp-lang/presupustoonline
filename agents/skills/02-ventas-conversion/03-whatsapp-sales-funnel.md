# WhatsApp Sales Funnel para Agencia de Viajes

## Estrategia de Comunicación por WhatsApp

### Mensaje Automático Post-Submit
```
¡Hola [nombre]! 👋

Recibimos tu solicitud de viaje a [destino].
En las próximas horas un asesor especializado
te va a contactar con las mejores opciones.

Mientras tanto, ¿alguna preferencia especial
que quieras agregar?
```

### Plantillas de Seguimiento

**24h después** (sin respuesta):
```
Hola [nombre]! ✈️
Te escribo porque vi que solicitaste
presupuesto para [destino]. 

Ya estoy armando algunas opciones
con las mejores tarifas.
¿Seguís con ganas de viajar? 😊
```

**48h después** (segundo contacto):
```
[Hola/Disculpá] [nombre]! 🙌
Quería compartirte estas opciones
que armé para [destino]:

🔹 Opción A: [breve descripción] - $[precio]
🔹 Opción B: [breve descripción] - $[precio]

¿Te gusta alguna? Podemos ajustarla!
```

**7 días después** (último intento):
```
Hola [nombre]! Quería avisarte que
las tarifas de [destino] están
con disponibilidad limitada. 

Si querés, puedo reservarte sin cargo
mientras lo pensás. Avísame!
```

### Segmentación por Estado
- **No Cotizado**: Mensaje de bienvenida + confirmación de recepción
- **En Cotización**: "Estoy armando tu presupuesto"
- **Cotizado**: Propuesta enviada + seguimiento
- **Enviado Cliente**: "¿Pudiste ver la propuesta?"
- **Concretado**: Post-venta (documentación, recomendaciones)
- **Cancelado**: "¿Podemos mejorar algo?"

### Objeciones Comunes y Respuestas

| Objeción | Respuesta |
|----------|-----------|
| "Estoy viendo" | "Entiendo, te dejo la info. ¿Te parece si te escribo el jueves?" |
| "Muy caro" | "Podemos ajustar. ¿Hotel más sencillo o menos días?" |
| "Ya encontré otra opción" | "Te entiendo. ¿Viste que incluimos [beneficio diferencial]?" |
| "No sé fechas fijas" | "Sin problema, trabajamos con fechas flexibles. Te muestro opciones" |

## Buenas Prácticas para WhatsApp Comercial
- Responder siempre dentro de las primeras 2 horas (lead caliente)
- Usar emojis con moderación estratégica
- Enviar audios solo si el cliente los usa
- NO spammear (máximo 1 mensaje cada 24h sin respuesta)
- Incluir link al detalle del presupuesto web
- Cerrar siempre con una pregunta abierta
