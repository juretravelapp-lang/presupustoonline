# Diseño de Imágenes Promocionales para Viajes

## Especificaciones Técnicas para Banners

### Imágenes del Carrusel (Hero Banner)
- **Resolución base**: 1920×800px (2.4:1 aspect ratio)
- **Responsive**:
  - Desktop: 1920×800px
  - Tablet: 1200×500px
  - Mobile: 800×400px
- **Formato**: WebP (moderno) + JPEG fallback
- **Calidad**: 80-85% (balance calidad/peso)
- **Peso máximo**: 200KB por imagen

### Composición Visual Recomendada
```
┌─────────────────────────────────────────────────┐
│                                                   │
│                                                   │
│   [ZONA SEGURA - TEXTO]                           │
│   Título: 2 líneas max                           │
│   Subtítulo: 1 línea                             │
│   CTA: visible                                   │
│                                                   │
│   ────────────── 60% altura ─────────────────    │
│                                                   │
│   Imagen de fondo: destino, colores vibrantes     │
│   Gradiente overlay: 60% opacidad                 │
│   De abajo hacia arriba (oscuro a claro)          │
└─────────────────────────────────────────────────┘
```

### Estilo Visual de las Promos
- **Overlay**: Gradiente lineal `rgba(0,0,0,0.6)` → `rgba(0,0,0,0.1)` de abajo arriba
- **Texto**: Blanco, sombra de texto sutil para legibilidad
- **Posición texto**: Abajo a la izquierda o centrado
- **Elementos decorativos**: Línea dorada delgada separando título/subtítulo
- **Badge de precio**: Fondo dorado semitransparente, texto oscuro

## Assets Gráficos Adicionales
- **Iconos de destinos**: Emoji nativos (🇧🇷, 🏝️, 🇪🇺, 🇺🇸, 🇲🇽, 🚢)
- **Favicon**: Versión simplificada del logo (32×32, 16×16, 192×192)
- **OG Image**: 1200×630px para redes sociales (cuando se comparte link)
- **Trust badges**: Escudo de seguridad, "Sin compromiso", "Mejor precio"
