# Diseño para Redes Sociales - Travel Jure

## Estilo Visual para Redes
- **Tono**: Aspiracional, luminoso, cálido
- **Paleta**: Consistente con marca (azul #1E3A5F + dorado #C9A96E)
- **Tipografía**: Playfair Display (títulos), Inter (cuerpo)
- **Estilo foto**: Viajeros felices, destinos paradisíacos, detalles de viaje

## Formatos por Plataforma

### Instagram (Canal Principal)
| Formato | Dimensiones | Uso |
|---------|-------------|-----|
| Post cuadrado | 1080×1080px | Destinos, promos, tips |
| Post horizontal | 1080×566px | Paisajes, destinos panorámicos |
| Story | 1080×1920px | Ofertas flash, testimonios, polls |
| Reel | 1080×1920px (vertical) | Videos cortos de destinos |
| Carrusel | 1080×1080px (múltiples) | Guías, tips, casos de éxito |

### Contenido Sugerido
| Tipo | Frecuencia | Ejemplo |
|------|------------|---------|
| Destino destacado | 2/semana | "Brasil en enero: playas y carnaval" |
| Tips de viaje | 1/semana | "5 tips para ahorrar en tu vuelo" |
| Testimonial | 1/semana | "La experiencia de María en Europa" |
| Oferta/Promo | 1/semana | "All Inclusive Caribe desde USD X" |
| Detrás de escena | 1/quincena | "Así armamos tu viaje perfecto" |
| Usuario real | 1/quincena | Repost de cliente viajando (UGC) |


### Plantillas de Post
**Plantilla Destino**:
- Imagen hero del destino (80% espacio)
- Nombre destino: Playfair Display, blanco, bold
- Precio desde + CTA: "Solicitá tu presupuesto"
- Línea dorada decorativa

**Plantilla Tips**:
- Fondo azul oscuro (#1E3A5F)
- Texto dorado (#C9A96E) para números
- Texto blanco para contenido
- Iconos alineados
- Marca sutil (logo pequeño abajo)
